# previsaoProblema

Previsão simples baseada nos últimos X meses(podendo escolher quantos meses para trás) tomando como valores esses ultimos meses escolhidos para prever quais serão a quantidade e a probabilidade de ocorrer tais problemas nos meses Y seguintes(também por escolha).
Logo, é uma previssão razoavelmente realista, usando modelo de Poisson de probabilidade, considerando as tendencias e dados do passado, para "prever" os seguintes, o que a torna confiável para 1 ou 2 meses seguintes, para prazos mais longos de 6 meses deve se considerar outros fatores estruturais e decisões críticaws da empresa para avaliar os resultados.
Código feito em R, para análise de dados, dos quais foram pegos de uma planilha google sheets da empresa Ansal, tais dados sao de problemas dos onibus da cidade de Juiz de Fora MG(tomando em conta o tipo de problema e sua data de ocorrencia).
